🌐 Rony Ghosh — Personal Developer Portfolio

A fully responsive personal portfolio website built from scratch using pure HTML5, CSS3, and Vanilla JavaScript — no frameworks, no templates, no dependencies. Every line of code was written by hand to showcase my skills, projects, experience, and achievements in a clean and professional format.

🎯 Project Overview

This portfolio was built to present my professional identity as a Full Stack Developer in the most impactful way possible. Instead of using a website builder or pre-made template, I coded everything from scratch to demonstrate my real front-end development skills. The design follows a light editorial aesthetic with strong typography, smooth animations, and a navy blue and orange color palette that communicates professionalism and energy. The layout adapts seamlessly across desktops, tablets, and mobile devices.

✨ Features

Custom Animated Cursor — Dot-and-ring cursor with a smooth lagging effect using requestAnimationFrame

Scroll Progress Bar — Gradient bar at the top that fills as the user scrolls through the page

Scroll Reveal Animations — Sections animate into view using the native IntersectionObserver API

Animated Skill Bars — Proficiency bars animate from zero to target width on scroll

Active Nav Highlighting — Navigation links highlight automatically based on the current section

Mobile Hamburger Menu — Full-screen overlay menu for smooth navigation on smaller screens

Frosted Glass Navbar — Sticky navigation with a backdrop-filter blur effect

Hero Stats Card — Displays CGPA, project count, certifications, and tech skills at a glance

Project Cards — Both projects shown in a consistent split layout with tech stack chips and live links

Experience Timeline — Vertical animated timeline presenting internship and education history
